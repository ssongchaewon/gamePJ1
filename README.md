## 🧩 게임 설명

**플레이 방식**

- 🐚 마우스로 위치를 조정하고 클릭해 해산물을 떨어뜨리세요.  
- 🪸 같은 종류의 해산물끼리 부딪히면 더 큰 해산물로 **합체!**  
- 🎯 합체할수록 점수가 올라가며, 고래까지 키워보세요!  
- 💣 어항 속 해산물들이 **빨간 선을 넘으면 게임오버!**  
- 🧠 최고 점수는 **자동 저장(LocalStorage)** 으로 관리됩니다.
